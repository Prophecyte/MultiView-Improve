How to Build the Desktop App

MultiView uses Electron to wrap the web app into a native desktop experience. To activate the download buttons, you need to:

  1. Install Node.js (v18+) and npm
  2. Create an electron/ folder in the project root
  3. Run npm init and install electron + electron-builder
  4. Create a main.js that loads MultiView's URL in a BrowserWindow
  5. Run npx electron-builder --win --mac --linux to generate installers
  6. Host the builds (GitHub Releases, S3, etc.) and update the download links

Once builds are hosted, update this page's button hrefs to point to the download URLs. The buttons will automatically activate when links are provided.
